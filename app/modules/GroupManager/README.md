# GroupManager 模块

## 模块说明

群组管理模块，支持群组禁言、解禁、踢出、全员禁言、全员解禁、撤回消息等操作。

## 模块命令

### 基础管理命令
- 禁言：`ban` - 支持at和QQ号，支持批量操作
- 解禁：`unban` - 支持多个用户
- 踢出：`kick` - 踢出指定用户
- 全员禁言：`banall` - 禁言全体成员
- 全员解禁：`unbanall` - 解除全员禁言
- 撤回：`recall` - 撤回指定消息
- 封禁自己：`banme` - 随机禁言自己
- 禁言排行榜：`banrank` - 查看禁言记录

### 特殊标记消息功能（仅群管理和群主可用）
- `atall` - @全体成员并提醒关注该消息
- `settodo` - 将消息设置为群待办
- `setessence` - 将消息标记为群精华

**使用说明**：
- 直接发送包含关键词的消息：对当前发送的消息进行标记
- 回复某消息后发送关键词：对被回复的消息进行标记

**示例**：
- 直接发送 `settodo` - 将这条包含settodo的消息设为群待办
- 回复某条消息并发送 `settodo` - 将被回复的那条消息设为群待办

### 其他功能
- 警告未活跃用户：`警告未活跃用户 天数` - 例如：警告未活跃用户 30
