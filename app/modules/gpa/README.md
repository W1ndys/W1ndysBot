# GPA模块

## 功能

绩点查询模块，支持查询目标绩点在班级中的百分位排名。

## 命令

### 绩点百分比

查询目标绩点在指定班级和学期的百分位排名。

**用法：**
```
绩点百分比 班级名称 学期 目标绩点
```

**参数说明：**
- `班级名称`：班级名称，支持模糊匹配（如：22网安、24电子信息）
- `学期`：学期号，格式为 `xxxx-xxxx-x`，或使用 `all` 表示全部学期
- `目标绩点`：要查询的绩点值（0-5之间）

**示例：**
```
绩点百分比 22网安 2024-2025-1 3.91
绩点百分比 24电子信息 all 3.64
```

**返回信息：**
- 班级名称和学期
- 目标绩点
- 班级人数
- 实际排名（第X名）
- 超过比例（超过班级X%的同学）
- 最接近的记录绩点
- 班级平均绩点（参考）

## 注意事项

1. 查询结果基于数据库中已有的绩点排名数据，直接返回最接近目标绩点的实际记录
2. 班级名称请尽量准确，系统会尝试模糊匹配
3. 使用该功能需要先在群聊中开启 GPA 模块开关（需系统管理员权限）
