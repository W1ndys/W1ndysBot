# QFNUFreshmanQuestionQuery 模块开发指南    

该模块用于实现用户在QQ群内查询题库和获取题目答案的功能。以下是开发该模块的步骤和注意事项。

对于该群开关打开的群才生效

开关打开的群，通过匹配用户的消息内容，字符串匹配数据库中题库的题目内容，返回对应的四个选项，并使用emoji标出正确答案和错误答案。

使用回复类型消息回复

你需要自行编写测试脚本来读取数据库结构，以便于调试和验证功能的正确性。

该项目环境是uv管理的Python项目，请确保你熟悉uv的使用方法。

数据库文件位于模块目录根目录下，文件名为 freshman_questions.db
