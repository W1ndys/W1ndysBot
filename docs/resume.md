📋 简历项目描述

项目名称

W1ndysBot - 基于 OneBot v11 协议的多功能 QQ 机器人

---

一、项目简介（简短版）

基于 Python + WebSocket 开发的高性能 QQ 机器人，遵循 OneBot v11 协议标准，采用插件化架构设计，支持 24+ 功能模块动态加载。实现了完善的群管理、智能问答、违禁词检测、情感分析等功能，日均处理消息量数千条。

---

二、项目描述（详细版）

独立设计并开发了一款企业级 QQ 群管理机器人系统，基于 OneBot v11 协议标准，采用 Python 异步编程实现高并发消息处理。项目采用插件化架构，实现了模块热加载机制，支持 24 个功能插件的独立开发与部署。核心功能涵盖智能群管理、AI 违禁词检测、FAQ 问答系统、情感分析、二维码识别等，并集成飞书告警实现运维监控。

四、项目亮点 / 技术难点

1️⃣ 插件化架构设计

- 设计并实现了模块自动发现与热加载机制，基于 importlib 动态导入
- 插件通过统一接口 handle_events(websocket, msg) 接入，支持独立开发、即插即用
- 采用 asyncio.create_task() 实现模块间非阻塞并发处理，单个模块异常不影响全局

2️⃣ 高性能异步消息处理

- 基于 websockets 库实现与 QQ 客户端的持久化 WebSocket 连接
- 全链路异步设计，支持高并发消息处理，单实例可同时服务数十个群聊
- 实现消息定时撤回系统，支持持久化存储与断线恢复

3️⃣ 多级权限管理系统

- 设计了 4 级权限体系：系统管理员 → 群主 → 群管理员 → 普通成员
- 实现群级别 + 全局级别的模块开关系统，支持 SQLite 持久化
- 支持开关状态批量复制、失效群清理等运维功能

4️⃣ 智能内容审核

- 集成 OpenCV + pyzbar 实现群聊二维码检测与自动处理
- 基于 scikit-learn 实现消息情感分析，辅助群氛围管理
- 实现关键词过滤 + AI 智能违禁词检测双重防护

5️⃣ 完善的 API 封装层

- 封装了 1900+ 行的 OneBot API 调用层，覆盖消息、群组、用户管理等全部功能
- 设计了消息段生成器，支持文本、图片、@、语音、合并转发等消息类型的便捷构建
- 实现了批量踢人、批量禁言等高级群管理 API

6️⃣ 运维监控体系

- 基于 loguru 实现日志轮转、压缩归档、异步写入
- 集成飞书 Webhook，实现机器人掉线告警与状态监控
- 实现心跳检测与自动重连机制
